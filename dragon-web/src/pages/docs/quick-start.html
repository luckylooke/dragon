DRAGON quick start

Dragon library is possible to install as npm package and use it as ES6 module or as ES5 UMD module, or even it can be loaded as <script> into the simple page.

Installation

  • ES6 module
  • ES5 module
  • No module 
npm install dragon
or 
yarn dragon
then in your project just import dragon and start using 
import dragon from 'dragon'
let dragonInstance = dragon(element, options)
